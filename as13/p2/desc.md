### 問題内容

```
class Integer:
    def __init__(self, v):
        '''
        your code here
        '''

    def add(self, i):
        '''
        your code here
        ''' 

a = int(input())
b = int(input())
a_i = Integer(a)
b_i = Integer(b)
sum_i = a_i.add(b_i)
print(sum_i.v)
```

#### 入力例1
```
15
56
```

#### 出力例1
```
71
```

654

```

```
