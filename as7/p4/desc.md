### 問題内容
正の整数を入力する．  
入力が`奇数`だった場合，リスト`[0, 1]`を，入力の数だけ要素に持つリストを出力せよ．  
入力が`偶数`だった場合，リスト`[0, 1]`の要素を，入力の数だけ要素に持つリストを出力せよ．  

###ヒント

### 入出力例
#### 入力例1
```
3
```

#### 出力例1
```
[[0, 1], [0, 1], [0, 1]]
```

#### 入力例2
```
4
```
#### 出力例2
```
[0, 1, 0, 1, 0, 1, 0, 1]
```

- ファイル名およびコード内（コメントも含む）に全角文字は使わない
