### 問題内容
文字列を`,`（カンマ）で2回まで分割しリストにせよ、もし大文字のアルファベットが含まれていれば小文字に変換し、そのリストを出力するプログラムを記述せよ。

### 入出力例
#### 入力例1
```
AbcD,Efgh,IJkl,mnopq
```

#### 出力例1
```
['abcd', 'efgh', 'ijkl,mnopq']
```

#### 入力例2
```
top,MIDDLE,boTTom
```
#### 出力例2
```
['top', 'middle', 'bottom']
```

### 注意事項

- ファイル名およびコード内（コメントも含む）に全角文字は使わない  
- ローカル環境（自分のPC）で実行結果が合っていることを確認してから提出する
