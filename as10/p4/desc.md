### 問題内容
整数`a`，`b`を受け取り，`a`の`b`乗を出力するプログラムを記述せよ．  
ただし，`**`演算子，`math.pow()`関数，`for`文，`while`文は使わずに，再帰関数を使うこと．

### 入出力例
#### 入力例1
```
-2
5
```

#### 出力例1
```
-32
```

#### 入力例2
```
3
0
```

#### 出力例2
```
1
```

#### 入力例3
```
2
-2
```

#### 出力例3
```
0.25
```

### 注意事項

- ファイル名およびコード内（コメントも含む）に全角文字は使わない  
- ローカル環境（自分のPC）で実行結果が合っていることを確認してから提出する
