変更したあとはコメントアウトを外すこと．

```
a = int(input())
b = int(input())

def add(a, b):
    '''
    your code here
    '''

print(add(a, b))
```


### 入出力例
#### 入力例1
```
1
2
```

#### 出力例1
```
3
```

- ファイル名およびコード内（コメントも含む）に全角文字は使わない
